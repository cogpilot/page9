Core Concepts

  • Service Worker as Kernel: Interprets configuration files and manages runtime behavior
  • Everything is a File: Plan9 philosophy - all resources accessed through file-like interfaces
  • Namespace Mounting: Virtual file system with configurable mount points
  • Web Workers: Process isolation through dedicated worker contexts
  • Zero Backend: Fully self-contained, static deployment on GitHub Pages
  • Configuration-Driven: Runtime behavior defined through page9.config.json
